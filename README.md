# Assamese & English Digital Lexicon 📖

A modern, ultra-fast, and open-access bidirectional digital dictionary designed for seamless cross-lingual lookups between Assamese and English. Built entirely on an independent serverless edge framework to manage a massive vocabulary archive with near-zero latency.

🌍 **Live Application:** [assamesedictionary.org](https://assamesedictionary.org/)

---

## ✨ Key Features

* **Massive Word Database:** Fast semantic mapping across an extensive dataset of **293,000+ words**.
* **Advanced Vocabulary Mapping:** Deep definitions, comprehensive synonyms, and structural sentence usage examples for complex English terminology.
* **Bidirectional Search:** Instantly lookup terms from either Latin strings or native Assamese script blocks.
* **Serverless Edge Performance:** Powered by global key-value memory caching so queries return instantly without standard relational database bottlenecks.
* **Offline Access (PWA):** Fully installable on your mobile home screen or desktop with service-worker progressive storage to work reliably anywhere.
* **Interactive Synonyms:** Clickable vocabulary tokens allowing fluid structural discovery inside definitions.
* **Community-Driven Logging:** Integrated translation and missing-entry correction submission drawers.

---

## 🛠️ Built With

* **Frontend:** Clean Vanilla JavaScript (ES6+), HTML5, and production-optimized Tailwind CSS.
* **Runtime API Engine:** Cloudflare Workers running high-performance serverless logic.
* **Global Data Layer:** Cloudflare KV (Key-Value) Cache Store.
* **Hosting Pipeline:** Cloudflare Pages for optimized global asset edge-delivery.
